Network Classifications
  • LAN: Local Area Network
  • MAN: Metropolitan Area Network
  • WAN: Wide Area Network

    • Network Models
  • Bus Network: A network which nodes are directly connected to a duplex link called a bus.
  • Star Network: A LAN which all nodes are Personal Computers.

    • Protocols
  • CSMA/CD Ethernet: Carrier Sense Multiple Access / Collision Detection detects and describes how Ethernet communicates among nodes
  • CSMA/CA Wifi: Carrier Sense Multiple Access / Collision Avoidance detects and describes how to avoid collisions in communications between PCs

    • Combining Networks
  • Repeater: A device that relays a signal, amplifies the signal, then rebroadcasts the signal.
  • Bridge: A device that connects two LANs together
  • Switch: A device that connects several network devices such as computers, in order to recieve, give, or hold data
  • Router: A network device that forwards data between computer networks
  • Gateway: A device that directs traffic in and out of a specific network.

    • Client/server model
  • P2P: Peer-to-peer
  • Distributed System: A collection of computers that are connected by a central computer network.
  • Cloud Computing: The delivery of data over the internet.

    • Internet Addressing
  • Tier 1: A network that can reach every other network on the Internet
  • Tier 2: A network that engages in the practice of peering with other networks
  • Tier 3: A network that engages primarily with end customers, cable, or DLS

    • IP Address
  • ICANN:Internet Corporation for Assigned Names and Numbers
  • DNS: Domain Name System, turns domain name into an IP address
  • TLD: Top-level Domain
  • Contry-code TLDs: Represent specific geographic locations

    • Internet Protocols
  • FTP: File Transfer Protocol is used to transfer files between computers
  • HTTP: Hypertext Transfer Protocol is used to transfer data over the web
  • SMTP: Simple Mail Transfer Protocol is used for sending e-mail over the internet

    • World Wide Web
  • URL: Uniform Resource Locator is the specific location of a web resource
  • Client-Side Activities: An operation that is preformed by the client in a computer network.
  • Server-Side Activities: An operation that is preformed by the server in a computer network.

    • Internet Four-Layer Software
  • Application Layer: A layer that specifies that communication protocols used by the host in a computer network.
  • Transport Layer: A layer that provides end to end communication services for applications.
  • Network Layer: A layer that is part of the communication connection with the internet.
  • Link Layer: The lowest layer that confined the link that a host connected to.

    • Internet Is:
  • Packet-Switching Network: A form of computer comm. that sends data in small packets.
  • Fault Tolerant: A system that handles failure of computer hardware ot software.
  • Redundant: Duplicate devices that are used as backups.

    • Cybersecurity
  • Malware: Software intentionall designed to cause harm to a system.
  • Virus: A computer program that replicates itself by modding other computer programs.
  • Worm: A computer program that affects that functions of both hardware and software.
  • Trojan Horse: Any malware that misleads users of its true intent.
  • Spyware: Malware that tries to keep itself hidden while it secretly records info.
  • Phishing: An attempt to trick a person into revealing info.
  • DDoS Attack: Distributed Denial-of-Service tagets websites and servers.

    • Protection
  • Firewall: A network security system that controls incoming and outgoing traffic.
  • Spam Filter: A program that detects unwanted emails and prevents them from getting into the users inbox.
  • Proxy Server: A server application that acts as the intermediary between client and the resourse.
  • Antivirus Software: Computer software that prevents and removes malware.
  • Cryptography: A method to transform messages to make them secure.
  • Public Key Encryption: A form of cryptosystem in where, decryption and encryption is preformed
